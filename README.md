# single-page_cv

A single-page CV (Curriculum Vitae) built with semantic HTML as part of the roadmap.sh frontend developer learning path.

## Project URL

https://roadmap.sh/projects/single-page-cv

## Description

This project is a professional CV webpage created using only HTML. It demonstrates:

- Semantic HTML structure
- SEO meta tags for search engine optimization
- Open Graph (OG) tags for social media sharing
- Proper document structure with sections for skills, education, and experience
- Accessibility best practices
- Mobile-responsive design
- Professional styling with CSS

## Features

- **Semantic HTML5**: Uses proper tags like `<header>`, `<main>`, `<section>`, `<article>`, and `<address>`
- **SEO Optimized**: Includes meta description, keywords, and author tags
- **Social Media Ready**: Open Graph tags for better sharing on social platforms
- **Favicon**: Custom favicon included
- **Single Page Layout**: Clean, organized sections for easy navigation
- **Professional Design**: Modern styling with border accents and color scheme

## Sections

1. **Header**: Name, job title, and contact information
2. **Skills**: Technical skills and competencies
3. **Education**: Academic background and achievements
4. **Experience**: Work history with detailed achievements and skills
5. **Social Links**: LinkedIn and GitHub profiles

## Technologies Used

- HTML5
- CSS3
